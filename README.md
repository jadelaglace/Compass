原始文字（人类）：
这是一个以“我”为核心，高度结构化且动态演进的个人知识宇宙蓝图。

以下是对个人知识库体系结构的清晰梳理。

核心引擎：动态相关度评分系统

这是整个知识库的最高准则，如同一个引力场，决定了每个知识元素的“价值”和“位置”。

*   评分维度:
    1.  我最感兴趣的（现在）： 当前的热情所在，正在投入时间精力的领域。
    2.  我最该关心的（未来）： 面向未来的战略布局，长期目标和潜在的发展方向。
    3.  共识（过去）： 已经被验证的、基础性的、构成你认知基石的知识。

*   运作方式:
    *   综合打分： 知识库中的任何元素（无论是学科、知识点还是感悟）都会被赋予一个基于以上三个维度的综合分数。
    *   动态更新： 这个分数不是一成不变的，需要定期回顾和调整，以反映你认知重心的迁移。
    *   历史追踪： 系统会记录分数的变化历史，形成你的“认知演化图谱”，让你清晰地看到自己兴趣与重心的变迁。

️ 第一大界：知识的宇宙地图（架构层）

这是你知识世界的宏观结构，一个从基石到分支的金字塔。所有在此结构中的学科和方向，都受核心引擎的评分系统调控。

*   第一层：基石系列（世界观维度）
    *   这是你认知世界的根本框架，是最高维度的分类。
    *   时间系列： 涵盖时间、历史、科学史、哲学史等。
    *   空间系列： 涵盖地理、坐标系统、宇宙学等。
    *   物质系列： 涵盖从宏观宇宙、恒星到微观粒子、肉身（生物学）等。
    *   意识系列： 涵盖心理学、认知科学、自我、思维模型、模拟理论等。

*   第二层：学科系列（领域维度）
    *   在基石之上建立的成熟知识领域。
    *   示例： 数学、物理、化学、英语、商务、社会学、金融学等。

*   第三层：分支系列（专业维度）
    *   学科下的具体细分方向，是知识探索的最前沿。
    *   示例： 高等数学、离散数学、高分子化学、机器学习、渲染工程、相对论、艺术史等。

第二大界：知识的原子与标本（内容层）

这是知识的具体承载单元，分为理论（知识层）与实践（案例层），两者相互印证，形成闭环。

知识层（理论原子）

这是经过你高度浓缩和消化的知识单元。

*   标题： 知识的名称。
*   标签： 从第一大界的架构中梳理出的标签，可以跨学科、跨层级，不唯一。例如，一个关于“博弈论”的知识，可以同时拥有 #社会学 #数学 #意识系列 等标签。
*   来源（可选）： 知识的出处。
*   内容高浓缩脑图： 核心！用模型、公式、清单、流程图等形式，将知识提炼成最精华的部分。例如：SMART原则解析图、细分市场五力模型、色温与饱和度关系图。
*   内容正文： 对脑图的详细解释和补充。

案例层（实践标本）

这是知识层理论在现实世界中的应用。

*   标题： 案例的名称。
*   标签： 同样使用统一标签系统，便于与知识层关联。
*   应用场景： 描述案例发生的背景和环境。
*   案例正文： 详细记录案例的过程、结果和分析。

相互引用关系：
*   在“知识层”的条目下，可以引用一个或多个“案例层”的条目，作为理论的实践佐证。
*   在“案例层”的条目下，可以反向链接到其所应用的“知识层”理论，阐明其背后的原理。

第三大界：我的认知轨迹（日志与感悟层）

这是完全属于“我”的、带有强烈个人色彩的思考记录，是知识库中最具生命力的部分。

日志层（时间切片）

按时间维度和重要程度记录事件与想法。

*   长期的： 高度凝聚的、方向性的、里程碑式的重要节点记录。
*   中期的： 经过压缩和概括的阶段总结。
*   短期的： 重要的片段标记，不做压缩，保留原始信息。
*   实时的： 临时、待整理的原始记录，是知识的“收集箱”。

感悟层（思想结晶）

这是经过你深度思考后产生的原创内容，是你智慧的直接体现。

*   形式： 可以是只言片语的灵感火花，也可以是大概的思路框架，或是需要完整留存的成熟思考。
*   核心： 必须是“自己写的”，是你与知识互动后的产物。

全局互联：标签与引用

*   统一标签系统： 三大界的所有内容都通过一套统一的标签系统进行连接。这些标签源于第一大界的架构，但可以灵活地应用于任何内容上，打破层级和领域的壁垒。
*   双向引用： 不仅“知识”与“案例”之间，日志、感悟与架构、知识、案例之间，都应建立良好的相互引用关系。例如，一篇“感悟”可以引用某个“知识”作为论据，也可以链接到某条“日志”作为背景。

这个体系结构清晰地勾勒出了一个以你为中心、动态生长、理论与实践结合、记录与思考并重的个人知识宇宙。它既宏大又具体，既有严谨的结构又保留了个人思考的灵动性。
