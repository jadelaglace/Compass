# Compass · 罗盘

> 个人认知操作系统 · Personal Compass OS

---

## 产品定位

**Compass（罗盘）** —— 让高价值内容自然浮现，让过时内容优雅衰减。

不是"第二大脑"，不是知识仓库。是**认知操作系统**：帮你记住、排序、浮现最重要的知识。

---

## 核心价值

**三维评分引擎** × **飞书Bot** × **Agent原生**

## 开发阶段

```
Phase 1 (MVP · 8周)
├── Vault 结构设计
├── 三维评分引擎
├── 飞书Bot
├── Obsidian 移动端接入
└── Agent API

Phase 2（向量语义层）
└── FAISS 向量索引 + 语义搜索

Phase 3（可视化层）
└── Web UI + D3.js 图谱
```
