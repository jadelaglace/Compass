# Compass 1.0 Demo Archive

状态：已归档，不再进行主动功能开发。

此目录保留 Compass 1.0 的完整 Git 历史、分支、远端配置和本地工作树，作为：

- 运行参考。
- 选择性能力拆分的来源。
- 新系统迁移后的回归比较对象。

Compass 2.0 从干净的新根目录开始，不继承本仓库历史。其全局文档暂存于同级目录 `Compass-2.0-docs`，新系统建立后迁入独立的 `compass-docs` 仓库。

归档时本地工作树包含未提交改动。这些改动被原样保留，未为了归档而自动提交、重置或丢弃。`v1.0-demo-final` 标签标记归档时的已提交基线；检查实际归档状态时同时查看 `git status`。
