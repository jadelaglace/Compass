# 上世纪无AI时代的群体行为预判与舆情分析工作

上世纪无AI、计算机未普及的年代，这类群体行为预判、舆论趋势分析、社会/市场推演的工作并非不存在，只是完全以人工为主导，靠经验研判、小范围实证、定性分析完成，没有量化模拟和自动化工具，核心是"用人力替代算力"，且对应职业更偏向传统调研、分析岗，工作方法和技术手段与当下差异极大。

## 一、核心对应职业

| 领域 | 职业 |
|------|------|
| 政府/智库 | 调研干事、舆情信息员、政策研究员 |
| 企业 | 市场调研专员、公关顾问、行业分析师 |
| 社科领域 | 社会调查研究员、传媒评论员 |
| 信访/综治岗 | 民情信息员 |

## 二、核心工作方法（五步流程）

1. **信息采集**：全线下/人工搜集，入户走访、纸媒剪报、信访记录
2. **数据处理**：纸笔+计算器，纯人工统计分类
3. **研判分析**：定性为主，靠经验做趋势推演
4. **输出结论**：人工撰写报告，对策偏宏观
5. **反馈跟踪**：线下持续走访，人工调整对策

## 三、支撑学科与理论

- 传播学经典理论（议程设置、沉默的螺旋）
- 群体动力学理论
- 经典社会调查理论
- 系统动力学（定性分析）
- 经典博弈论
- 描述性统计学

## 四、上世纪 vs 现在核心差异

| 维度 | 上世纪 | 现在（AI工具） |
|------|--------|---------------|
| 核心依赖 | 人工经验、小范围实证 | 算力+模型+大数据 |
| 数据规模 | 极小，纸质零散数据 | 海量，全渠道数字化 |
| 分析方式 | 纯定性，无模拟 | 定量建模+多智能体模拟 |
| 效率 | 数周/数月 | 数小时 |
| 精准度 | 偏宏观，无精准风险点定位 | 可精准定位风险点和传播路径 |

**本质差异：** 上世纪是"用脚跑数据，用脑子凭经验判趋势"；现在是"用机器挖数据，用模型模拟趋势，用人经验做校验"。
